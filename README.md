# fajarakupunkturhomecare

Website statis untuk GitHub Pages.

## Cara publish ke GitHub Pages

1. Buat repository baru di GitHub, misalnya `fajarakupunkturhomecare`.
2. Upload isi folder ini ke repository tersebut.
3. Pastikan file `index.html` ada di root repository.
4. Buka `Settings` -> `Pages`.
5. Pada `Build and deployment`, pilih:
   - `Source`: `Deploy from a branch`
   - `Branch`: `main`
   - `Folder`: `/ (root)`
6. Simpan, lalu tunggu beberapa saat sampai GitHub memberi URL website.

## Catatan

- File `.nojekyll` disertakan agar GitHub Pages menyajikan file statis apa adanya.
- Halaman utama ada di `index.html`.
- Website ini memakai gambar dan data dari URL eksternal, jadi koneksi internet tetap diperlukan saat dibuka.
